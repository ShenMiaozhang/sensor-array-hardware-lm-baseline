# Sensor Array Hardware and Baseline LM Solver

This repository is the public release for reproducibility in our TII submission.  
It provides the hardware design files and a baseline magnetic localization solver so other researchers can reproduce results and benchmark future methods.

## Repository contents

- **Hardware/**: sensor array PCB schematics and fabrication files.
- **Software/**: MATLAB baseline solver (magnetic dipole model + LM optimization).
- **tii-articles-2501.pdf**: manuscript draft/reference file.

## Purpose

1. Share a clean baseline implementation for fair comparison.
2. Provide complete hardware artifacts for replication.
3. Support future benchmarking and follow-up research.

For software usage details, see `Software/README.md`.
